## Installation

### Clone

- Clone this repo to your local machine:

```bash
git clone https://github.com/YFears/Project-11.git
```

### Using
- Use these commands to build, develop or deploy project

```bash
npm run build
```

```bash
npm run dev
```

```bash
npm run deploy
```
